# Video Playback System

## ⚠️ LOOPING ONLY HERE

Looping is allowed ONLY in playback

---

## LOOP LOGIC

IF video ends:
    set frame position = 0

---

## CONTROLS

- SPACE → pause/play
- LEFT → -2 sec
- RIGHT → +2 sec
- Q → quit

---

## RESULT

- Smooth looping playback
- No impact on logging
